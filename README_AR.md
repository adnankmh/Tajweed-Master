# Tajweed Master Pro V11 — Flutter Enhanced

نسخة Flutter محسّنة قبل بناء Android، تحتوي على:

- مصحف بنظام السور + الأجزاء + الصفحات + الفهارس.
- رقم الصفحة والجزء يتم حسابهما باستخدام حزمة quran لضبط بيانات الصفحات والأجزاء.
- تلوين أحكام التجويد على الحروف نفسها بطبقة رسم فوق النص بدون تقطيع الكلمات.
- مشغل تلاوات فيه 10 قرّاء، منهم قرّاء معلمون مثل المنشاوي المعلم والحصري المعلم.
- صفحة التلاوات تعمل بالإنترنت عبر روابط MP3، ويمكن لاحقًا إضافة التحميل دون إنترنت بعد مراجعة الحقوق.
- تفعيل اختيار الخط وحجم الخط على التطبيق والمصحف.
- امتحانات بأسئلة أكثر، وأسئلة إضافية توليدية من مكتبة الأحكام، وشرح مفصل للأخطاء بعد الامتحان.
- زر تثبيت التطبيق PWA من الإعدادات، مع تعليمات Android/iPhone.
- لوحة إدارة تظهر فقط بعد تسجيل دخول المدير:

- يمكن للمدير إضافة أسئلة وشروحات محليًا على الجهاز/المتصفح.

## رفع التحديث على GitHub

1. فك الضغط.
2. ادخل داخل المجلد حتى ترى `pubspec.yaml` و `lib` و `assets` و `web`.
3. افتح GitHub > Repository > Code > Add file > Upload files.
4. حدّد كل الملفات داخل المجلد وارفعها.
5. Commit مباشرة على main.
6. راقب Actions حتى تصبح خضراء.
