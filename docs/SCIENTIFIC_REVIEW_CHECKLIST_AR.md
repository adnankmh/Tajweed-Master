# قائمة مراجعة علمية قبل النشر الرسمي

- التأكد من سلامة نص القرآن حرفيًا.
- مراجعة الرسم العثماني مع مصدر موثوق.
- مراجعة كل تلوين آية بآية وحرفًا حرفًا.
- مراجعة أمثلة الأحكام.
- مراجعة الشروحات والأسئلة.
- إضافة أسماء المراجعين أو جهة الاعتماد إن وجدت.
- إضافة تنبيه واضح أن التطبيق تعليمي إذا لم يكن معتمدًا رسميًا.
